# Vscode 快捷键

1. **打开 vscode 的快捷键菜单**

    ```javascript
    Cmd + K + R
    ```

2. **在当前行下插入空行**

    ```javascript
    Cmd + Enter
    ```

3. **在当前行上插入空行**

    ```javascript
    Cmd + Shift + Enter
    ```

4. **打开 safari 浏览器的开发者工具**

    ```javascript
    Cmd + Opt + I
    ```

5. **快速选择一个单词**

    ```javascript
    Shift + Opt + 左右键
    ```
