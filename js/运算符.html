# 十进制和二进制的转换

1 << 2 // 相当于把十进制的数字转换成二进制操作： 00000001 => 把 1 左移了两位 00000100，就是 4
4 & 4 // 4  "&"是二进制的运算符 与，会把每一位对比，【都等于 1 才是 1】，100 & 010 => 0  100 & 001 => 0  前面的 4 & 4 就相当于 100 & 100 => 100

// parseInt(string, radix) string 十进制表示的字符串，radix 指定的基数（指定字符串是几进制），最终返回十进制
parseInt('1000', 2) // 8 ，把一个二进制字符串 1000 转成 十进制
(20).toString(2) // 10100，把十进制转成二进制
